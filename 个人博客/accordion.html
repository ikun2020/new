一个超简单的jQuery手风琴代码

March 22，2017


只需要几行代码即可实现的手风琴效果。需要引入jQuery库。插件用于PC端，个人制作，仅供参考。


html部分

<div id="hcc">
 <ul>
  <li class="p1" style="background:#4285f4">
   <a href="javascript:">
    <p>G</p>
   </a>
  </li>
  <li class="p2" style="background:#ea4335">
   <a href="javascript:">
    <p>O</p>
   </a>
  </li>
  <li class="p3" style="background:#fbbc05">
   <a href="javascript:">
    <p>O</p>
   </a>
  </li>
  <li class="p4" style="background:#34a853">
   <a href="javascript:">
    <p>G</p>
   </a>
  </li>
  <li class="p5" style="background:#ea4335">
   <a href="javascript:">
    <p>L</p>
   </a>
  </li>
  <li class="p6" style="width: 600px; background:#fbbc05">
   <a href="javascript:">
    <p>E</p>
   </a>
  </li>
 </ul>
</div>


CSS部分

					
*{
margin: 0px;
padding: 0px;
font-family:"微软雅黑";
font-size: 20px;
text-decoration: none;
}
#hcc{
width: 1200px;
height: 400px;
margin: 100px auto;
}
#hcc ul li{
list-style-type: none;
width: 100px;
height: 400px;
float: left;
}


jQuery部分

					
$(function(){
 $("#hcc ul li").mouseover(function(){
 $(this).stop(true).animate({width:"600px"},500)
 .siblings().stop(true).animate({width:"100px"},500)
});
});
